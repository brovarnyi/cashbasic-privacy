cashBASIC — 개인 재무 관리 앱

개요

cashBASIC은 사용자의 개인정보 보호를 최우선으로 설계되었습니다. 이 앱은 완전히 오프라인으로 작동하며, 외부 서버로 개인 데이터를 수집, 저장 또는 전송하지 않습니다.

데이터 저장

모든 재무 데이터(계좌, 거래, 카테고리, 예산)는 Apple의 SwiftData 프레임워크를 사용하여 기기에 로컬로 저장됩니다. 데이터는 절대 기기를 벗어나지 않습니다.

수집하지 않는 데이터

  • 개인 정보(이름, 이메일, 전화번호) 수집 없음
  • 사용 분석 또는 추적 없음
  • 위치 데이터 수집 없음
  • 기기 식별자 수집 없음
  • 쿠키 또는 유사 기술 사용 없음

제3자 서비스

cashBASIC은 제3자 SDK, 분석 서비스, 광고 네트워크 또는 외부 API를 통합하지 않습니다.

아동 개인정보 보호

이 앱은 13세 미만 아동을 포함하여 누구로부터도 의도적으로 데이터를 수집하지 않습니다.
